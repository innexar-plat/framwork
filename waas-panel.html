Exemplo: Integração Hotmart
Quando cliente comprar no Hotmart, dispare este webhook para criar o tenant automaticamente:
// Hotmart webhook → seu backend → WaaS API

const res = await fetch("/api/v1/integration/workspaces", {
  method: "POST",
  headers: {
    "X-Api-Key": process.env.WAAS_KEY,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    external_id: hotmart.buyer.email,
    name: hotmart.buyer.name,
    plan_code: "pro",
    niche_code: "imobiliaria"
  })
});
